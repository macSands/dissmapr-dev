# dissmapr: Workflow for Compositional Dissimilarity and Biodiversity Turnover Analysis

Tools to compute and map biodiversity turnover at the macroscale.
Provides functions for generating spatial grids, retrieving
environmental data, computing multi-order zeta diversity, fitting
I-spline models for generalised dissimilarity, predicting community
turnover, and delineating and mapping bioregions and their changes over
time.
